# Educational VR Logic Gates Simulator
An interactive, educational Virtual Reality (VR) application made with Unity for Oculus/Meta headsets to teach users the fundamental mechanics of basic logic gates (AND, OR, XOR).

This project was developed from scratch as part of the 'Virtual Reality' course, part of the Master's Degree in Computer Graphics, Games and Virtual Reality at Rey Juan Carlos University (URJC).

The main objective of this application is to translate abstract digital electronics concepts into a tangible, highly interactive 3D environment, to make it easy for kids to learn this concepts. Users interact with a control console featuring input buttons and dynamic feedback systems to visualize how electrical currents propagate through different logic circuits.

### Key features for an optimal user experience
**- Dynamic Visual Feedback:** Utilizes a custom emissive shader to animate the flow of electric current in real-time from inputs to the logic gates.
**- Spatial sound:** Integrates spatial audio (such as a low buzzing sound paired with current movement of the electrical current) to reinforce user interaction and presence.
**- Haptic & Tactile Feedback:** Leverages Meta/Oculus controller haptics to provide physical buzz/vibration feedback whenever the user interacts with the buttons or pieces, enhancing tactile realism.

### Game Modes
**1. Structured Levels**  

In the first three levels, the logic gate is fixed to the test bench. The user experiments with inputs to observe the gate's behavior and successfully illuminate the terminal light bulb.

- AND Gate Level.
- OR Gate Level.
- XOR Gate Level (Hidden): Locked by default. To unlock this level, the user must explore the room and discover a hidden gate component inside a closet. This mechanic was specifically designed to reward exploration and spatial curiosity.

**2. Free Mode** 

After completing or discovering the main gates, the user unlocks a free-form sandbox. In this mode, users can manually select different logic gates from inventory buckets and swap them onto the test bench to dynamically test and contrast their behaviors.
